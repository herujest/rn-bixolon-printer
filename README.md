# rn-bixolon-printer

native mobile implementation on bixolon printer

## Installation

```sh
npm install rn-bixolon-printer
```

## Usage

```js
import { multiply } from "rn-bixolon-printer";

// ...

const result = await multiply(3, 7);
```
